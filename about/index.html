项目经验

Kffmpeg - 基于 FFmpeg 的自定义异常推流

将难以在功能测试里回归的用例，利用 FFmpeg 整合为工具，配合自动化框架 QT4S 实现腾讯云直播的异常监控。

  • 自定义 SEI 信息写入、检查
  • 基于 已发送时间、已发送 AVPacket 数的 dts/pts 跳变
  • 指定时间段禁用 音频/视频 数据的发送
  • 支持命令行调用、 HTTP 调用两种调用方式
  • 将以上功能用 C++ Boost Crow 框架整合

    权限管理、短信广播接收、短信生成&分析、定时发送短信、异常上报（Bugly）
